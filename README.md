# e-plantShopping: Paradise Nursery E-Commerce Application

Welcome to **e-plantShopping**, a fully dynamic React and Redux-based shopping cart application built for **Paradise Nursery**—an online destination offering a diverse variety of houseplants. 

This application features a rich user interface, seamless state management, and real-time cart updates, providing a fluid shopping experience for plant enthusiasts.

## 🌿 Key Features

### 🏡 Landing Page
* A welcoming, visually appealing entry screen featuring a clear introduction to the Paradise Nursery brand.
* An interactive **"Get Started"** button that seamlessly transitions users directly into the product catalog.

### 🧭 Navigation Bar
* A globally persistent header allowing quick transitions between the **Landing Page**, **Product Listing**, and **Shopping Cart**.
* An integrated **Cart Icon** displaying a live, dynamic count of the total items currently added to the cart.

### 🪴 Product Listing Page
* Organizes an extensive plant inventory into distinct botanical classifications, including:
  * **Aromatic Plants** (e.g., Lavender, Jasmine)
  * **Medicinal Plants** (e.g., Aloe Vera, Mint)
* Each plant is presented in a customized structural card containing high-quality images, nomenclature, pricing, descriptions, and contextual **"Add to Cart"** buttons that automatically disable when items are added.

### 🛒 Dynamic Shopping Cart Page
* A detailed ledger showcasing chosen items with responsive line-item summaries.
* Each selected item displays a thumbnail preview, itemized unit cost, subtotal calculation, and clear adjustments to increase, decrease, or completely delete item quantities.
* Includes **"Continue Shopping"** to return to the catalog and a placeholder **"Checkout"** mechanism alerting users that functionality is coming soon.
